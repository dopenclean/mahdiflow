# 3D Particle System with Hand Tracking

A real-time interactive 3D particle system built with Next.js, Three.js, and MediaPipe.

## Features

- **Hand Tracking**: Control particles with your hands using the webcam.
  - **Scale**: Move hands apart to scale the system.
  - **Explode**: Clench fists to make particles explode/expand.
- **Templates**: Switch between Heart, Flower, Saturn, Buddha, and Fireworks shapes.
- **Color Control**: Customize particle colors.
- **Modern UI**: Sleek, glassmorphism-inspired interface.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.
4. Allow camera access when prompted.

## Technologies

- **Next.js**: Framework
- **Three.js**: 3D Graphics
- **MediaPipe Hands**: Hand Tracking
- **Tailwind CSS**: Styling
- **Lucide React**: Icons
